HandleAllErrorAttribute Class

Represents an attribute that is used to handle an exception that is thrown by an action method.

Unlike the HandleErrorAttribute, this attribute handles all types of exceptions.

Examples

Usage:

Change the ~/App_Code/FilterConfig.cs as following:

Copy
public static void RegisterGlobalFilters(GlobalFilterCollection filters)
{
  filters.Add(new HandleAllErrorAttribute());
}

In web.config, add the customErrors element to the system.web element:

Copy
<configuration>
  <system.web>
    <customErrors mode="On" />
  </system.web>
</configuration>
