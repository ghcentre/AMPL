StringBetweenOptions Enumeration

Specifies various options for the Between(String, String, String, StringBetweenOptions, StringComparison) method.

This enumeration has a FlagsAttribute attribute that allows a bitwise combination of its member values.

Remarks

The Between(String, String, String, StringBetweenOptions, StringComparison) default behavior is:

  • If at least one substring specified by the start and end parameters is not found in the source string, the method return the empty string.
  • The substrings specified by the start and end parameters are not included in the result substring.
